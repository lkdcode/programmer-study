FAB (Floating Action Button) 용도

우측 하단 동그라미 버튼이 어떤 역할을 하면 좋을까요?

A

Quick Actions (빠른 작업)

클릭하면 펼쳐지면서 "새 링크 추가", "프로필 편집", "미리보기" 같은 바로가기 제공

B

Help / Support (도움말)

클릭하면 도움말, FAQ, 피드백 보내기 등 지원 메뉴 표시

C

Theme / Settings (테마/설정)

다크모드 토글, 테마 변경 등 빠른 설정 접근

D

다른 용도가 있어요

터미널에서 직접 설명해주세요

FAB 예시 미리보기
