HDDC Global Features

글로벌 기능 범위를 먼저 정해봅시다

현재 HDDC는 링크인바이오 서비스입니다. 글로벌 기능들이 필요한 대상 페이지는 어디인가요?

A

Dashboard 계열 (관리 페이지)

/dashboard/edit, /dashboard/analytics 등 — 사이드바 + 헤더로 관리 도구 느낌

B

마케팅/랜딩 페이지

/, /auth 등 — 방문자 경험 개선, CTA 버튼

C

둘 다 — 전체 앱

모든 페이지에 걸친 글로벌 네비게이션 시스템

참고: 현재 구조
현재 HDDC 라우트 구조
/ ─────────────── 홈 (랜딩페이지)
/auth/login ───── 로그인
/auth/signup ──── 회원가입
/dashboard/edit ─ 프로필 편집 ← 현재 개발중
/[username] ───── 공개 프로필
